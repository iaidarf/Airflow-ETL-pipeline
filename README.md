## Airflow ETL-pipeline

### Цель 

Необходимо настроить ежедневный расчет ключевых метрик работы приложения за прошедший день и их запись в базу Clickhouse.

### Этапы реализации

В ноутбуке (**ETL_DAG.ipynb**) проделываю все этапы:
- подключение к базе для получения необходимых данных
- выполняю необходимые преобразования данных
- рассчитываю метрики
- проверяю полученный результат

Переношу код в  Python‑скрипт (**ETL_DAG.py**), добавляя работу с библитоекой airflow для работы с DAG-ом.

### Результат
Автоматизация ежедневной загрузки расчитанных ключевых метрик приложения в базу данных.
